Export & Import Data
The CSV must include a header row with these column names. Required columns must be present; Optional columns can be omitted.
Column Required Format Example
date ✓ Yes YYYY-MM-DD 2025-05-05
set_number ✓ Yes Integer 1
reps ✓ Yes Integer ≥ 1 20
duration_seconds Optional Integer 45
logged_time Optional HH:MM:SS or HH:MM AM/PM 08:30:00
id Optional Unique string 1714900800000
💡 Import merges — existing records are never overwritten. Rows with matching id values are skipped as duplicates.
💡 logged_time controls where the set appears in your daily timeline. If omitted, sets default to 12:00 PM.
